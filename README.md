# ✦ Endsinger AI ♡

A tiny public home for Endsinger AI releases and its Dalamud update feed.

Dalamud custom repository URL:

```text
https://raw.githubusercontent.com/endsingersena/endsinger-ai-releases/main/repo.json
```

Endsinger AI provides gear guidance, crafting preparation, current-job detection, and read-only inventory material counts. It does not execute movement, combat, crafting actions, or simulated input.
